# VideoClipEditViewTest
使用android原生的MediaExtractor,MediaCodec和opengl对视频进行编辑, 裁剪,滤镜, 特效等.

####  功能点
1. 视频播放  (不使用ffmpeg)   
2. 视频裁剪    
3. 视频帧预览   
4. 裁剪区间动态可选择.   
5. 美颜滤镜
6. opengl特效. 特效叠加后视频生成   


####  使用
1. 必须是android 17 及以上版本
